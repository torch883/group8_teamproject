# Game Statistics Analyzer

A Python application to analyze player game statistics.

## Team Members
- Igor Kukhta — Models (player.py)
- Arman Mukashev — File handling (file_handler.py), readme.md
- Temirlan Nabi — Statistics (stats.py)
- Daniyal Anarov — Main, utils, tests

## How to Run

1. Clone the repository
2. Make sure Python 3.x is installed
3. Run the program:

python main.py

## How to Run Tests

python -m unittest tests/test_stats.py

## Features
- View leaderboard
- View average scores per player
- View best overall and daily performance
- Add new game sessions
- Filter stats by player
